# DYLoader - 抖音与小红书无水印视频下载器

一个极简、精美、极速的抖音和小红书视频无水印在线下载工具，支持视频与音频提取。

## ✨ 功能特性

- 🎵 **双平台支持**：同时支持抖音（Douyin）和小红书（Xiaohongshu）
- 📥 **无水印下载**：提取最高画质的无水印视频源
- 🎶 **音乐提取**：支持提取抖音视频的背景音乐（MP3）
- 🔗 **短链支持**：自动跟踪解析 `v.douyin.com` 及 `xhslink.com` 短链
- 🔄 **流式代理**：后端代理下载，解决跨域限制，强制触发浏览器下载对话框
- 🌙 **精美暗黑界面**：毛玻璃 + 霓虹渐变光晕的 Glassmorphism 视觉风格

## 📦 技术栈

- **后端**：Node.js + Express
- **HTTP 请求**：Axios
- **前端**：原生 HTML5 + CSS3 + JavaScript

## 🚀 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动服务器

```bash
npm start
```

### 3. 打开浏览器访问

```
http://localhost:3000
```

## 📋 使用方式

1. 打开任意抖音或小红书视频
2. 复制分享链接（支持长链接和手机分享的短链接）
3. 粘贴到输入框并点击**「提取并解析」**
4. 点击下方的下载按钮，即可保存无水印视频或音频

## 🌐 支持的链接格式

| 平台 | 链接示例 |
|------|---------|
| 抖音（Web） | `https://www.douyin.com/video/xxxxxx` |
| 抖音（短链） | `https://v.douyin.com/xxxxxx` |
| 小红书（Web） | `https://www.xiaohongshu.com/explore/xxxxxx` |
| 小红书（短链） | `https://xhslink.com/xxxxxx` |
